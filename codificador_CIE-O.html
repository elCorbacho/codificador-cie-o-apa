Directrices oficiales — §4.1 Tabla 14 del PDF

Reglas de codificación A–K

El PDF establece 10 reglas oficiales (no existe Regla I, intencionalmente). Cada regla aplica a situaciones específicas de codificación.

Reglas de topografía
A
Localizaciones mal definidas
Si el diagnóstico no especifica tejido de origen, usar el tejido específico del índice alfabético — no C76 (SAI).
Carcinoma del brazo → C44.6 (piel), no C76.4
B
Prefijos (peri-, para-, pre-...)
Localización modificada por prefijo no listado → C76 (mal definida), salvo que el tipo tumoral indique tejido específico.
Tumor peripancreático → C48.0 (retroperitoneo)
C
Tumor que rebasa límites
Tumor en dos o más subcategorías contiguas sin punto de origen determinable → subcategoría .8
Carcinoma esofágico cervicotorácico → C15.8
D
Linfomas — código topográfico especial
Ganglionares → C77._. Extraganglionares → sitio de origen. Múltiples regiones → C77.8. Sin localización → C80.9.
Linfoma MALT gástrico → C16.9 · Linfoma ganglionar → C77.9
E
Leucemias — siempre C42.1
TODAS las leucemias → C42.1 (médula ósea). Excepción: sarcoma mieloide (9930/3) → localización del depósito.
LMA → C42.1 · Sarcoma mieloide en ganglio → C77.9
Reglas de morfología
F
Sistema matricial de comportamiento
Usar comportamiento correcto aunque el término exacto no aparezca en CIE-O. La matriz permite cualquier combinación. El patólogo tiene la última palabra.
Cordoma benigno → 9370/0 (aunque no esté listado)
G
Grado: asignar el MAYOR
Si el diagnóstico indica dos niveles de grado, usar el mayor. Los dígitos de linaje 5–8 prevalecen sobre diferenciación 1–4.
Mod. dif. con áreas pobremente dif. → grado 3
H
Morfología asociada a localización
Usar código topo sugerido entre paréntesis si el diagnóstico no indica localización. Si hay localización indicada, usarla aunque difiera del código sugerido.
Basocelular sin loc. → C44._ · Con loc. distinta → usar esa
J
Diagnósticos morfológicos compuestos
Si el término compuesto no aparece, invertir el orden de las raíces de las palabras.
Mixofibrosarcoma → buscar Fibromixosarcoma (8811/3)
K
Múltiples términos morfológicos
Si un tumor tiene dos adjetivos con códigos distintos y no hay código único, usar el numéricamente mayor (generalmente más específico).
Carcinoma escamoso de células transicionales → 8120/3

Casos especiales

El PDF establece reglas específicas para situaciones que no siguen el patrón general.

Tumores del SNC — Grados OMS vs CIE-O (§4.6, Tabla 27)

Los grados OMS I–IV para tumores del SNC no son equivalentes al sexto dígito de CIE-O. El grado OMS estima pronóstico; el sexto dígito de CIE-O describe diferenciación. Si no hay descripción verbal del grado → usar 9.

Laboratorio AP vs Registro de cáncer (§4.3.2)

El PDF distingue dos contextos de uso con reglas diferentes para el comportamiento:

Laboratorio AP

Rastreo de muestras. Permite /6 (metástasis) y /9 (incierto). Ej: biopsia de ganglio con adenocarcinoma metastásico → C77.0, 8490/6

Registro de cáncer

Solo registra el tumor primario. Usa únicamente /2 (in situ) y /3 (maligno). El /6 se deduce de la base del diagnóstico.

Ejemplo (Tabla 19 del PDF):
a. Biopsia ganglio: C77.0 8490/6
b. ★ Primario gástrico: C16.1 8490/3 ← solo este va al registro
c. Metástasis bronquial: C34.1 8490/6
NIC III / CIN III y displasias (§4.3.2)

Uno de los errores más frecuentes en codificación anatomopatológica:

NIC III / CIN III → comportamiento /2 (in situ)

Se considera equivalente al carcinoma in situ. El PDF indica que el comportamiento correcto es /2, no /3.

8077/2NIC III / CIN III cervical (C53._)
8077/2NIVA III / VAIN III vaginal (C52._)
8077/2NIV III / VIN III vulvar (C51._)
8077/2NIA III / AIN III anal (C21.1)
Si el anatomopatólogo considera que la NIC III no es equivalente a carcinoma in situ, puede cambiar a /1 (Regla F).
Neoplasias primarias múltiples (§4.4) — Reglas IARC/IACR 2004
Regla 1

El reconocimiento de dos o más cánceres primarios no depende del tiempo transcurrido entre diagnósticos.

Regla 2

Un cáncer primario es el que se origina en una localización o tejido primario y no es extensión, recidiva ni metástasis.

Excepciones sistémicas

Sarcoma de Kaposi (9140) y tumores hematopoyéticos se cuentan una sola vez por persona, independientemente de cuántos órganos afecten.
